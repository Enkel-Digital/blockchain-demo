# Blockchain Demo
A simple interactive frontend app to demo concepts used by blockchain technology.


## Basic commands
Get started by installing dependencies
```shell
npm i
```

Run in development mode
```shell
npm run serve
```

Build for production deployment
```shell
npm run build
```
